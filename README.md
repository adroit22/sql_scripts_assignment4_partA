# sql_scripts_assignment4_partA

run the script inside create_db.sql file (This script will create the Conference_Review Database).
### Step2 -

run the scripts in following order to insert data into the tables - 
insert_author.sql, 
insert_paper.sql,
insert_reviewer.sql,

insert_topic.sql
